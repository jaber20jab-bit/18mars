# 📋 ملخص المشروع

## 🎯 الهدف الرئيسي

إنشاء موقع إلكتروني شخصي احترافي مجاني على GitHub Pages لنشر المقالات والمذكرات، مع تصميم داكن أنيق ومريح للعين، وتركيز خاص على تجربة القراءة المريحة للمقالات الطويلة.

## 📊 نظرة عامة على المشروع

| العنصر | التفاصيل |
|--------|----------|
| **اسم المشروع** | مدونتي الشخصية |

| **الاستضافة** | GitHub Pages (مجاني) |
| **التصميم** | داكن ومريح للعين |
| **اللغة** | العربية |
| **الجمهور المستهدف** | القراء والمهتمين بالمحتوى العربي |

## 🎨 نظام التصميم

### الألوان
```
الوضع الداكن:
- الخلفية: #1a1a2e (كحلي داكن)
- المحتوى: #16213e (كحلي متوسط)
- النص: #eaeaea (رمادي فاتح)
- التمييز: #e94560 (أحمر وردي)

الوضع الفاتح:
- الخلفية: #f5f5f5 (رمادي فاتح)
- المحتوى: #ffffff (أبيض)
- النص: #333333 (رمادي داكن)
```

### الخطوط
- **العناوين**: Amiri (خط عربي أنيق)
- **النص**: Noto Sans Arabic (خط عربي واضح)
- **الأكواد**: Fira Code (لعرض الأكواد)
