# 📚 مدونتي الشخصية - منصة النشر الاحترافية

## 🎯 نظرة عامة على المشروع

موقع إلكتروني شخصي مجاني مخصص لنشر المقالات والمذكرات، يتميز بتصميم داكن أنيق ومريح للعين، مع التركيز على تجربة القراءة المريحة للمقالات الطويلة.

## ✨ المميزات الرئيسية

### 🎨 التصميم
- **تصميم داكن (Dark Mode)** - مريح للعين خاصة أثناء القراءة الطويلة
- **تصميم متجاوب** - يعمل على جميع الأجهزة (موبايل، تابلت، لابتوب)
- **خطوط عربية أنيقة** - استخدام خطوط Amiri و Noto Sans Arabic
- **مساحات بيضاء مدروسة** - لتحسين تجربة القراءة

### 📖 ميزات القراءة
- **وضع القراءة المريح** - تبديل بين الوضع الداكن والفاتح
- **شريط تقدم القراءة** - يظهر نسبة تقدمك في قراءة المقال
- **وقت القراءة المقدر** - يعرض الوقت المطلوب لقراءة كل مقال
- **حفظ التقدم** - يتذكر أين توقفت في القراءة
- **تكبير وتصغير الخط** - للتحكم في حجم النص

### 🔍 التنظيم والبحث
- **فهرس المقالات** - عرض جميع المقالات في صفحة واحدة
- **نظام التصنيفات** - تصنيف المقالات حسب الموضوع
- **بحث فوري** - البحث في جميع المقالات
- **تصفية حسب التاريخ** - ترتيب المقالات حسب تاريخ النشر

### 📱 تجربة المستخدم
- **قائمة تنقل سهلة** - وصول سريع لجميع الأقسام
- **زر العودة للأعلى** - للتنقل السريع في الصفحات الطويلة
- **مشاركة المقالات** - مشاركة سهلة عبر وسائل التواصل
- **طباعة محسنة** - تصميم مخصص للطباعة

## 🎨 نظام الألوان

### الوضع الداكن (افتراضي)
- **الخلفية الرئيسية**: `#1a1a2e` (كحلي داكن)
- **خلفية المحتوى**: `#16213e` (كحلي متوسط)
- **النص الرئيسي**: `#eaeaea` (رمادي فاتح)
- **النص الثانوي**: `#a0a0a0` (رمادي متوسط)
- **اللون المميز**: `#e94560` (أحمر وردي)
- **روابط**: `#0f3460` (كحلي فاتح)

### الوضع الفاتح (اختياري)
- **الخلفية الرئيسية**: `#f5f5f5` (رمادي فاتح جداً)
- **خلفية المحتوى**: `#ffffff` (أبيض)
- **النص الرئيسي**: `#333333` (رمادي داكن)
- **اللون المميز**: `#e94560` (أحمر وردي)

## 🔤 الخطوط المستخدمة

- **العناوين**: Amiri (خط عربي أنيق)
- **النص الرئيسي**: Noto Sans Arabic (خط عربي واضح)
- **الأكواد**: Fira Code (لعرض الأكواد البرمجية)

## 📄 صفحات الموقع

### 1. الصفحة الرئيسية (index.html)
- شريط ترحيب شخصي
- أحدث المقالات (3-5 مقالات)
- إحصائيات سريعة (عدد المقالات، التصنيفات)
- رابط للفهرس الكامل

### 2. صفحة المقال (article.html)
- عنوان المقال
- تاريخ النشر ووقت القراءة
- التصنيفات
- محتوى المقال
- أزرار المشاركة
- مقالات ذات صلة

### 3. صفحة الفهرس (archive.html)
- قائمة جميع المقالات
- تصفية حسب التصنيف
- ترتيب حسب التاريخ
- بحث في العناوين

### 4. صفحة التصنيفات (categories.html)
- قائمة التصنيفات
- عدد المقالات في كل تصنيف
- رابط لكل تصنيف

### 5. صفحة من أنا (about.html)
- نبذة شخصية
- روابط التواصل
- الصورة الشخصية

## 📱 التصميم المتجاوب

### نقاط التوقف (Breakpoints)
- **موبايل**: أقل من 768px
- **تابلت**: 768px - 1024px
- **لابتوب**: أكثر من 1024px

### تعديلات التصميم
- **الموبايل**: قائمة هامبرغر، عمود واحد
- **التابلت**: عمودان، قائمة مبسطة
- **اللابتوب**: تصميم كامل، قائمة أفقية
